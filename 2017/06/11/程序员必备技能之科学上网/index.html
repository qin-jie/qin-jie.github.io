程序员必备技能之Google一下

发表于 2017-06-11 | |

作为一个整天与代码打交道的程序员来说。遇到bug或者其他问题之后都需要百度一下。但是很多问题光靠百度并不能很好的解决。而且,正所谓 内事不决问百度, 外事不决问谷歌。很多时候我们都需要访问这个在国内并不存在的网站- Google。

常用方法

在国内访问谷歌等一些被屏蔽的网站一般都需要搭梯子, 就是科学上网服务。一般有好几种方法来实现。

1.免费VPN 或者 收费VPN

一是网上有很多这些服务的提供商。比较常见的比如的 GreenVPN、红杏、 赛风、开眼、 各种网络加速器 等。这些服务一般是提供给你每天2小时左右免费体验。可以付费成为永久会员体验。不过也有一些是免费提供的无限制服务。但是这些服务提供商并不保险, 万一商家跑路或者被封就没得用了。而且政府打压这些VPN也越来越厉害, 所以不太推荐使用

2. Host文件

Hosts是一个没有扩展名的系统文件，可以用记事本等工具打开，其作用就是将一些常用的网址域名与其对应的IP地址建立一个关联“数据库”，当用户在浏览器中输入一个需要登录的网址时，系统会首先自动从Hosts文件中寻找对应的IP地址，一旦找到，系统会立即打开对应网页，如果没有找到，则系统会再将网址提交DNS域名解析服务器进行IP地址的解析。
通过修改系统内的host文件来实现，这种方法最简单, 直接复制文件就可以，但是由于各个人使用的带宽种类，地区，被墙的程度不一样，所以有些地区使用本Hosts文件可能仍然无法使用Google。修改hosts文件具有一定的时效性，且视每个人的网络情况结果会有所不同。

3. 利用虚拟主机

还有一种就是利用自己购买的VPS主机自己搭建SS服务来实现科学上网。这种方式灵活 而且也可以利用VPS搭建WordPress博客服务 建站等。

这里主要介绍一下利用VPS搭建科学上网服务。

常见的VPS服务商一般有 搬瓦工、Vultr、 DigitalOcean、Linode、 Virmach。

VPS一般分为KVM VPS和OpenVZ VPS。KVM VPS相对好点, 价格也贵一点. 关于VPS详细的知识可以参考VPS新手入门全攻略

4. 为路由器安装ShadowSocks服务

首先需要给路由器刷OpenWRT系统。然后安装Shadowsocks服务。这个方法比较复杂。但是是一劳永逸的。给路由器加上这层Buff之后就不知道其他的客户端软件了。具体方法请 Google 一下
