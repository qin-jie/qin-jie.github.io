最近两三事

发表于 2017-08-22 | |

最近好久没有写博客了。 emmm…

主要原因还是自己太懒了。就在我突然想起来要写的时候。才发现最近电脑重装系统了。

不要手贱 🔪👋

在这里郑重的告诫一下自己。以后别为了一时新鲜就把自己的工作💻升级到预览版系统。

  • iOS工程打包上传时只能在运行正式版系统或者GM版系统的💻上进行。否则iTunes Connect会给你发📧提示你

  • 新系统虽然有各种新特性优化啥的。但是beta版系统一般很多坑。不建议在工作电脑上升级

  • 隔段时间必须备份个人数据。外置硬盘。Time Machine等

然而

升级到了10.13开发者预览版系统。结果发现不能App。而且各种无名卡顿。于是遂降级到10.12。然而Hexo博客文件夹忘记备份 😂 。然后就GG了。等想起来时已经晚了。

只能通过之前的html网页转换成md文件。再慢慢弄了。

所以以后不能在手贱了。
